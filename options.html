YT Outlier Finder Settings

Target Webhook

This webhook receives video data when you use the context menus on YouTube or the Send button in the popup. Paste the URL of your server/webhook endpoint.

Required: The vidIQ Vision for YouTube Chrome extension must be installed for reliable outlier scores (the extension scrapes the "X x" badges that vidIQ adds to video cards).

Payload your server will receive (POST, JSON):
[ { "videoURL": "https://www.youtube.com/watch?v=...", "title": "Video Title", "thumbnailUrl": "https://i.ytimg.com/...", "outlierMultiple": 12.5 // number or null } ]

Your server can respond with JSON containing a url (or similar) and the success toast will be clickable if present.
